# Project Charter

## Objective
Create an agent orchestration platform in which model-agnostic workers operate like employees, hand off structured context, and are overseen by a project analyst that produces executive-readable updates.

## Core capabilities
- Specialized worker agents
- Baton packet handoffs
- Durable semantic and episodic memory
- Model routing and escalation
- Analyst event summaries
- Local-first development, AWS-ready deployment

## Phase 1 success criteria
- Monorepo scaffolded
- Docker Compose stack runs locally
- FastAPI health endpoint works
- Next.js shell page loads
- PostgreSQL schema created
- Redis reachable
- Tests and linting run
