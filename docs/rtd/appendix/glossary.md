# Glossary

- **Workspace**: A registered project root under safe boundaries.
- **Task**: A unit of planned work.
- **Run**: A concrete execution attempt for a task/role.
- **Event**: Timeline record of project activity.
- **Baton**: Handoff payload between roles/stages.
- **Routing Decision**: Recommended next action.
- **Digest**: Analyst summary.
- **Context Bundle**: Assembled data for model input.
- **Context Reference**: Pointer to stored full content replaced in optimized context.
- **Autonomy Cycle**: One bounded iteration of staged autonomous execution.
