# TUI

Start with:

- `syncore tui`
- or `syncore open WORKSPACE`

## Views

- Dashboard
- Workspaces
- Tasks
- Task Detail
- Agent Runs
- Diagnostics
- Metrics

## Task Creation

In new task modal, provider/model selection is available, with model filtering and autocomplete behavior.

## Behavior

- Handles API offline state gracefully
- Periodic refresh polling
- Action shortcuts for route/digest/run controls
