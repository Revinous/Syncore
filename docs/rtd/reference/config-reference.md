# Configuration Reference

## Runtime Selection

- `SYNCORE_RUNTIME_MODE`: `native` or `docker`
- `SYNCORE_DB_BACKEND`: `sqlite` or `postgres`

## Database

- `SQLITE_DB_PATH`
- `POSTGRES_DSN`

## Redis

- `REDIS_URL`
- `REDIS_REQUIRED`

## Interface URLs

- `SYNCORE_API_URL`
- `NEXT_PUBLIC_API_BASE_URL`
- `ORCHESTRATOR_BASE_URL`
- `ORCHESTRATOR_INTERNAL_URL`

## Security

- `API_AUTH_ENABLED`
- `API_AUTH_TOKEN`
- `RATE_LIMIT_ENABLED`
- `RATE_LIMIT_WINDOW_SECONDS`
- `RATE_LIMIT_MAX_REQUESTS`

## Context Layering

- `CONTEXT_LAYERING_ENABLED`
- `CONTEXT_LAYERING_DUAL_MODE`
- `CONTEXT_LAYERING_FALLBACK_THRESHOLD_PCT`
- `CONTEXT_LAYERING_FALLBACK_MIN_SAMPLES`

- `AUTONOMY_DEFAULT_MODEL`
