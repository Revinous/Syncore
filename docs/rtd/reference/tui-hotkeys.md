# TUI Hotkeys

- `q`: quit
- `r`: refresh
- `d`: dashboard view
- `w`: workspaces view
- `t`: tasks view
- `a`: agent runs view
- `x`: diagnostics view
- `m`: metrics view
- `n`: new task modal
- `s`: scan selected workspace
- `o`: route next action for selected task
- `g`: generate digest for selected task
- `e`: execute/start run for selected task
- `v`: change view mode where supported
